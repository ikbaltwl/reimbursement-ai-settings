# Reimbursement AI Settings Prototype

Static legacy-style prototype for reimbursement policy settings, including the AI policy verification checked and unchecked states.

The form supports editable policy details, AI criteria popovers with searchable suggestions and custom keyword chips, add/remove benefit rows, currency and expiry dependencies, save/export actions, and a simulation modal.

## Run

```sh
python3 -m http.server 5175
```

Then open `http://127.0.0.1:5175`.
